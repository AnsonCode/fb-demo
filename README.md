## 用例

### 基本操作

- 增删改查：CURD查询，如分页查询
- 实时查询：将GET请求转变成准实时推送
- 订阅查询：接入消息队列等实时消息
- 关联查询：同数据源有外键关联查询
- 原生SQL：graphql无法覆盖的场景
    - SQL查询
    - SQL变更
- 关联创建：关联关系如何创建
